# reuters-LSTM-bidireccional

Este proyecto implementa un modelo de clasificación de texto basado en redes neuronales LSTM bidireccionales para clasificar noticias de Reuters en una de las 46 categorías posibles. Al utilizar capas LSTM bidireccionales, el modelo puede aprender patrones tanto en la dirección hacia adelante como hacia atrás, lo que mejora su capacidad para comprender el contexto del texto.
